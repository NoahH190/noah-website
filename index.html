What I've been working on

Hardware

MIPS Five-Stage Pipelined Processor

Verilog, Xilinx Vivado, C++, Python

Designed a synthesizable five-stage pipelined MIPS processor in Verilog, targeting the BASYS-3 FPGA with a frequency of >50MHz. The architecture features a custom memory controller using the AMBA AXI protocol, branch prediction, and hazard handling logic. Verification was conducted using constrained random testing with Cocotb and Xsim, alongside a custom C++ CPU emulator for register-write validation.

FPGA Memory Game

VHDL, Xilinx Vivado

Developed a VHDL-based memory pattern game controlled by a Mealy Finite State Machine. The system utilizes a Linear-Feedback Shift Register (LFSR) for pseudo-random sequence generation and interfaces with on-board peripherals like LEDs and push-buttons. Real-time gameplay is managed through custom comparator circuits and timeout logic to validate user inputs.

VGA Text Console on FPGA

Verilog, Xilinx Vivado

Engineered a resource-efficient VGA text console in Verilog, delivering real-time 640x480 resolution at 60Hz. The design features PLL-based 25.175 MHz clocking for stable signal integrity and optimized RTL for HSync/VSync generation, ensuring precise synchronization and minimal logic gate overhead.
